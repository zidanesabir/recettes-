# Welcome to recettes project


```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <GIT_URL>

# Step 2: Navigate to the project directory.
cd <PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev

# Cloud setup
# Frontend: Firebase Hosting (build output in dist)
# API: Firebase Function `api` (HTTPS). Default `VITE_API_URL` uses `/api` so Hosting rewrites route calls automatically.
```
